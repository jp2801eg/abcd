# Learning Plan: ABCD Community Asset Map MVP
### From Terminal Beginner → Working AI-Powered Tool
 
---
 
## The Goal
 
Build a standalone program that:
1. Accepts input two ways: **live typing during a community meeting** or **uploading notes, text, or spreadsheets afterward**
2. Uses Claude to identify and categorize assets across the 6 ABCD asset types
3. Outputs a **visual asset map** — a standalone HTML file that works offline in any browser
4. Allows **human editing** of the map to incorporate facilitator judgment, images, and design choices
 
The MVP is fully self-contained — no internet required to view maps once generated, no third-party platform dependency. It is designed so the underlying data layer can later be connected to platforms like Kumu or Miro without rebuilding the classification logic.
